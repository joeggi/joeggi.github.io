Corrupt Eldritch Order

Here is the website for my final project with the WolverineSoft team. This is a single-player, 2D-action platformer game. My team aimed to create an ultra polished game this semester, and so we took inspiration from GunStar Heroes, a simpler case study compared to some of our other projects.

You can find the Corrupt Eldritch Order steam page at this link.

  • Corrupt Eldritch Order

My work - Corrupt Eldritch Order

In this project, I worked as the QA Director, meaning I was resposible for fixing bugs in pretty much every area of the game. However, I specialized in feature implementation and testing of the player area of the game. This meant implementing mechanics such as the aiming system, weapon attributes, movement, dashing, and wall-jumping.
This game utilized the Unity game engine with C# scripting, and used Photon for multiplayer syncing.

Cellosseum

Here is the Steam page for my first project with WolverineSoft. This game is a bullet hell case study of Nova Drift, set in a faraway planet where a lone cell is fighting to start new life. Something unique about this game is that it was completed entirely remotely, with weekly meetings taking place fully on discord.

You can find the Cellosseum steam page at this link.

  • Cellosseum

My work - Cellosseum

In this game I was a part of the QA department on the encounter squad. I worked on creating a variety of enemies and their movement patterns, as well as fixing bugs related to enemy behavior.
This game utilized the Unity game engine with C# scripting.
